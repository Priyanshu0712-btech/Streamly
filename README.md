# 💬 Streamly

A modern real-time chat application built with the MERN stack and Stream Chat SDK. Streamly enables users to connect with friends, exchange messages instantly, customize the interface with multiple themes, and enjoy a responsive chat experience.

---

## 🚀 Live Demo

🔗 **Live Application:** [Live-demo](https://streamly-frontend-mxg0.onrender.com/)

---

## 🚀 Features

- 🔐 JWT Authentication
- 👤 User Onboarding
- 📝 User Profile Management
- 🤝 Friend Request System
- 💬 Real-Time One-to-One Chat
- ⚡ Instant Messaging using Stream Chat SDK
- 📱 Fully Responsive Chat UI
- 🎨 32 DaisyUI Themes
- 🌙 Theme Preference Persistence
- 🟢 Online User Status
- 📂 Conversation Sidebar
- 🔍 Clean & Modern UI
- ☁️ Cloud-Based User Data Storage

---

## 🛠️ Tech Stack

### Frontend
- React.js
- React Router
- Tailwind CSS
- DaisyUI
- Zustand
- TanStack Query (React Query)
- Axios
- Stream Chat React SDK
- Lucide React

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- bcrypt
- Cookie Parser
- Stream Chat Server SDK

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/Priyanshu0712-btech/Streamly.git
```

```bash
cd streamly
```

### Install dependencies

#### Backend

```bash
cd server
npm install
```

#### Frontend

```bash
cd client
npm install
```

---

## 🔑 Environment Variables

### Backend (`.env`)

```env
PORT=5000

MONGO_URI=your_mongodb_uri

JWT_SECRET=your_jwt_secret

STREAM_API_KEY=your_stream_api_key
STREAM_API_SECRET=your_stream_api_secret
```

### Frontend (`.env`)

```env
VITE_STREAM_API_KEY=your_stream_api_key
```

---

## ▶️ Run the Project

### Start Backend

```bash
cd server
npm run dev
```

### Start Frontend

```bash
cd client
npm run dev
```
